While, Do While, For

While

                        imagin if we want to write "console.log("Hello World!")" for 10 times. Now we can do it by repeating the code again and again. 
                        
                        But with the help of "while loop" we can print it as many times without repeating the code.

                        let counter = 1;

                        while(counter <= 10){
                            console.log("Hello World!")
                            // counter = counter +1; // = counter ++
                            
                        }

                        with - 

                        let counter = 10;

                        while(counter >= 1){    
                            console.log("Hello World!")
                            // counter = counter - 1; // = counter --
                   
                        }

Find the sum of all Even Numbers between 1 - 100

                        let counter = 1;
                        let sum = 0; 
                        while(counter <= 100){
                            if(counter % 2 == 0){
                                sum = sum + counter;
                            }
                            counter++;
                        }
                        document.write(sum)
                        console.log(sum)

Do While

do while will run at least once Even if statement is false at first place. 

                        let counter = 1;
                        
                        do{
                            console.log("Hello World!")
                            counter++;
                        }while(counter <= 10);

                        for(let counter = 1; counter <= 10 ;  counter++ ){
